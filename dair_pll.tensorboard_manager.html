dair_pll.tensorboard_manager

Interface for logging training progress to Tensorboard.

class dair_pll.tensorboard_manager.TensorboardManager(folder)[source]

Bases: object

Manages logging of the training process.

Given a set of scalars, videos, and meshes, writes to tensorboard.

Additionally, manages the spinning up and shutting down of a separate process to run the tensorboard server.
